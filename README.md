# Test task

## Steps

- install dependencies

```bash
npm i
```

- create a database (postgresql)

- create .env

```bash
cp sample.env .env

# fill in all the properties of .env
```

- to run it

```bash
npm run dev

# it will also create all tables in the given database
```
