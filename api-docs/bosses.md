# Bosses

## 1. Get

```http
GET http://localhost:8000/bosses
```

## 2. Get me

```http
GET http://localhost:8000/bosses/me
```

## 3. Update

```http
PUT http://localhost:8000/bosses
```

### JSON body

```json
{
    "firstname": "string",
    "lastname": "string",
    "email": "string"
}
```

## 4. Update

```http
DELETE http://localhost:8000/bosses
```
